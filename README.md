# My Portfolio Website

Portfolio website cá nhân được xây dựng với HTML, CSS, và JavaScript thuần.

## 🚀 Tính năng

- **Responsive Design**: Tương thích với mọi thiết bị (Desktop, Tablet, Mobile)
- **Modern UI**: Giao diện hiện đại, chuyên nghiệp
- **Smooth Animations**: Hiệu ứng mượt mà, thu hút
- **Contact Form**: Form liên hệ với validation
- **SEO Friendly**: Tối ưu cho công cụ tìm kiếm

## 📁 Cấu trúc thư mục

```
myportfolio/
│
├── index.html          # Trang chủ
├── css/
│   └── style.css      # File CSS chính
├── js/
│   └── main.js        # File JavaScript chính
└── README.md          # File hướng dẫn
```

## 🎨 Sections

1. **Hero Section**: Giới thiệu tổng quan
2. **About**: Về bản thân
3. **Services**: Các dịch vụ cung cấp
4. **Projects**: Danh sách dự án đã thực hiện
5. **Journey**: Hành trình phát triển
6. **Testimonials**: Đánh giá từ khách hàng
7. **Contact**: Form liên hệ

## 🛠️ Công nghệ sử dụng

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Google Fonts (Inter)

## 📝 Hướng dẫn sử dụng

1. **Mở website**: 
   - Double click vào file `index.html` để mở trong trình duyệt
   - Hoặc sử dụng Live Server extension trong VS Code

2. **Tùy chỉnh nội dung**:
   - Mở `index.html` và chỉnh sửa nội dung text
   - Thay đổi thông tin liên hệ (số điện thoại, email, social media)
   - Cập nhật các dự án và kinh nghiệm của bạn

3. **Tùy chỉnh màu sắc**:
   - Mở `css/style.css`
   - Chỉnh sửa các biến CSS trong `:root`:
     ```css
     :root {
       --primary-color: #4F46E5;
       --primary-dark: #3730A3;
       --secondary-color: #06B6D4;
       /* ... */
     }
     ```

4. **Thêm hình ảnh**:
   - Tạo thư mục `images/` 
   - Thêm hình ảnh của bạn
   - Thay thế các placeholder trong HTML:
     ```html
     <img src="images/your-photo.jpg" alt="Your Name">
     ```

## 🎯 Cá nhân hóa website

### Thay đổi thông tin cá nhân:

1. **Logo và Brand Name**:
   - Tìm class `logo` trong HTML
   - Thay "MyPortfolio" thành tên của bạn

2. **Hero Section**:
   - Cập nhật title và subtitle
   - Thay đổi description theo nghề nghiệp của bạn

3. **About Section**:
   - Viết lại phần giới thiệu về bản thân
   - Cập nhật thông tin kinh nghiệm

4. **Projects**:
   - Thêm/xóa dự án
   - Cập nhật tên, mô tả và công nghệ sử dụng

5. **Contact**:
   - Cập nhật số điện thoại: Thay `0123456789`
   - Cập nhật email: Thay `your.email@example.com`
   - Thêm link social media

## ⚡ Performance Tips

2. Sử dụng format WebP cho hình ảnh
3. Minify CSS và JavaScript khi deploy
4. Sử dụng CDN cho fonts
